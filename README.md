# Conversor de Base (Java WEB)

Projeto criado para a simples conversão de algarismos de e para várias bases numéricas, como binária, octal, decimal, hexadecimal, entre outras.

Projeto criado no NetBeans utilizando a linguagem Java voltada para WEB a partir das versões desenvolvidas em [C#](https://github.com/iagocolodetti/Conversor_de_Base "https://github.com/iagocolodetti/Conversor_de_Base") e [Java](https://github.com/iagocolodetti/Conversor_de_Base_Android "https://github.com/iagocolodetti/Conversor_de_Base_Android") para Android.

Aplicação criada com o padrão MVC, utilizando classe Java como model, página JSP como view (com Bootstrap 4 e JSTL) e Servlet como controller.
